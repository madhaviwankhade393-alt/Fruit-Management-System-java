# Fruit-Management-System-java

A CRUD-based Fruit Management System developed using Java, JDBC, and MySQL.

## ✨ Features

* Add Fruit
* View Fruits
* Update Fruit Details
* Delete Fruit

## 🛠️ Technologies

* Java
* JDBC
* MySQL
* Exception Handling
* OOP Concepts

## 🎯 Purpose

Developed to practice database connectivity and CRUD operations using Java and JDBC.
